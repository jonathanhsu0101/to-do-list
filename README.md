# What is this?

This is a simple To-Do list. Input your things to do and click "Add" one by one. Click "done" when it's done. Click "delete" when you want to remove.

# How to install?

$ git clone https://github.com/jonathanhsu0101/to-do-list.git

$ cd to-do-list

$ npm install

# How to run?

$ npm start
